# Security Policy

Ghost Writer is a local browser game prototype. It has no network server, authentication layer, or database.

## Supported versions

Security reports should target the latest `main` branch.

## Reporting a concern

Open a GitHub issue with a concise description. Avoid posting exploit details or private data in public issues.

Useful reports include:
- The affected file or script.
- A small reproduction case without private data.
- Expected vs. actual behavior.

## Scope

This is a game prototype. Security concerns would most likely relate to local file handling in test/CI scripts or the browser smoke test harness.

## Response expectations

Small project, limited capacity. Reports reviewed as maintainers are available.

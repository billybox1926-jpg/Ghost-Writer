Ghost Writer

A haunted typewriter noir adventure prototype.

Ghost Writer is a keyboard-controlled canvas game. Move with arrow keys, type command words, inspect clues with empty-line Enter, toggle generated audio with the Audio button, press Escape on an empty line to restart, and toggle Hardboiled Mode with empty-line F2.
Ribbon 100 percent. Hardboiled Mode off. Witness guarded. No letters typed. Find the True Name. Paper, ink, and wood will talk.

Move: Arrow keys while the typed line is empty

Type: letters and spaces, Enter to commit

Inspect: empty-line Enter near a clue

Toggle: empty-line F2 for Hardboiled Mode

Audio:

Restart: empty-line Esc; Esc while typing clears the line

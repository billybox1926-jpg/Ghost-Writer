Ghost Writer

A haunted typewriter noir adventure prototype.

Ghost Writer is a keyboard-controlled canvas game. Arrow keys move only when the typed line is empty. Letters and spaces type into the haunted typewriter. Enter inspects on an empty line or commits typed words. Escape clears typed text first, then restarts on an empty line. Empty-line F2 toggles Hardboiled Mode. Generated audio uses the on-screen Audio button.

First-run case notes
  • Keep the typed line empty to walk with the arrow keys. If letters are showing, press Escape once to clear them.
  • With an empty line, press Enter near paper, ink, wood, or Eddie to inspect and copy clues into the journal.
  • Start with the rain-blurred receipt, then inspect Eddie Pike. Stand close to Eddie and type REMEMBER, Enter, then ACCUSE, Enter.
  • After Eddie gives up the door word, type OPEN, Enter, then finish the case with MALLORY VALE, Enter.
  • Hardboiled Mode (empty-line F2) bars Backspace; in Mallory's slice, one near-witness REMEMBER costs ribbon but jumps Eddie straight to the door word.

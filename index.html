Ghost Writer

A haunted typewriter noir adventure prototype.

Ghost Writer is a keyboard-controlled canvas game. Move with arrow keys, type command words, inspect clues with empty-line Enter, toggle generated audio with the Audio button, restart with empty-line Escape, and toggle Hardboiled Mode with empty-line F2.
Ribbon 100 percent. Hardboiled Mode off. Witness guarded. No letters typed. Find the True Name. Paper, ink, and wood will talk.

Move: Arrow keys on an empty typed line

Type: letters, Enter to commit

Inspect: Empty Enter near a clue

Toggle: Empty-line F2 for Hardboiled Mode

Restart: Empty-line Esc
